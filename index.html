Introduction

Kyber est un schéma de chiffrement à clé publique post-quantique basé sur les Learning With Errors (LWE) sur les polynômes. Ce mini-projet illustre les étapes principales de Kyber, les concepts et quelques formules mathématiques simplifiées.

Principe mathématique

La clé publique est générée à partir d’un polynôme secret s(x) :

\[ A \cdot s + e = b \quad \text{où } A \text{ est public, } e \text{ est bruit} \]

Chiffrement

Pour chiffrer un message m :

\[ c_1 = A \cdot r + e_1, \quad c_2 = b \cdot r + e_2 + m \]

Déchiffrement

Le destinataire utilise sa clé secrète s pour retrouver m :

\[ m = c_2 - c_1 \cdot s \]

Exemple de code (pseudo-code)

def keygen():
    s = sample_secret()
    e = sample_error()
    A = public_matrix()
    b = A * s + e
    return (b, A), s

def encrypt(m, pk):
    b, A = pk
    r = sample_random()
    e1, e2 = sample_error(), sample_error()
    c1 = A * r + e1
    c2 = b * r + e2 + m
    return c1, c2

def decrypt(c, sk):
    c1, c2 = c
    return c2 - c1 * sk
